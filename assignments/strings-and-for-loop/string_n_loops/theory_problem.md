1. Explain commonly used string functions in Python.
    Explain the purpose of:
        - upper()
        - lower()
        - strip()
        - replace()
        - find()
<p>There are some commonly used string functions are split(),
islower(),isupper(),isdigit(),len().</p>
<h3>upper():-</h3>
<p>Converts all lowercase letters of a string into uppercase 
letters.</p>
<h3>lower():-</h3>
<p>Converts all uppercase letters of a string into lowercase 
letters.</p>
<h3>strip():-</h3>
<p>Removes extra spaces from the beginning and end of a 
string.</p>
<h3>replace():-</h3>
<p>Replaces a word or character with another word or
character.</p>
<h3>find():-</h3>
<p>Finds the position/index of a character or substring in a 
string and returns the index of the found element.</p>

2. Explain string slicing in Python.
        - Explain the syntax- string[start:stop:step]
        - Explain reverse slicing.
<p>String slicing in python is the process to cut a piece 
containing desired elements from the sring.</p>
<p>The syntax string[start:stop:step], start means the 
index from which we want to slice, stop means the index till 
which we want to slice and step means to make the slicing 
consecutive desired elements from start to stop</p>
<p>Reverse slicing in Python is used to read a string in 
reverse order using slicing syntax.</p>

4. Explain the difference between string indexing
and string slicing in Python.
<p>String indexing is used to access a single character of a 
string using its position number. String slicing is used to 
access multiple characters or a part of the string using a 
range of indexes. Indexing returns one character, while 
slicing returns a substring and can also use step values.
</p>
